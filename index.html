How to use these resources
1

Course Sequence Flowchart — reference flowchart. Click any course to see prerequisites, corequisites, and which courses it unlocks.

2

Curriculum Worksheet — advising audit tracker. Enter a student's name, toggle completed courses, track electives and GBD requirements, and save a personalized file.

3

Tags on course tiles show fall/spring availability, WIC status, and prerequisite bottlenecks. Scroll to the bottom of each map for critical path advising notes.

4

In the Curriculum Worksheet, use ⬇ Save Advisee File to download a fully populated copy for each student — no account or server required.
